🚀 Dev Environment Ready

This scaffold is running inside a Dev Container that works on both Windows and Linux hosts.

  • 🐍 Python – Flask app at localhost:5000
  • 🐘 PHP – built-in server at localhost:8000
  • 🌐 HTML / CSS – Live Server at localhost:5500
  • ⚛️ React + Vite – dev server at localhost:5173
  • ☁️ Cloudflare Worker – Wrangler at localhost:8787
  • 🤖 GitHub AI – see github-ai/ folder

🤖 GitHub AI Quick Start

Set your GitHub personal access token and run the demo to talk to GitHub Models (GPT-4o, Llama, Phi, Mistral, …) — all available free through the GitHub Marketplace. 

export GITHUB_TOKEN=ghp_your_token_here
python github-ai/github_models_demo.py

☁️ Cloudflare Integration

Deploy your React app to Cloudflare Pages or run Workers locally:

# React → Cloudflare Pages
cd react-cloudflare-app
npm run build
wrangler pages deploy dist

# Cloudflare Worker
cd cloudflare
wrangler dev
